Remarks
This Solve method overload determines whether a matrix value is a logical 1 or a logical 0 using the following default predicate:
Copy
private static Func<T, bool> DefaultPredicate<T>()
{
    return t => !EqualityComparer<T>.Default.Equals(t, default(T));
}
In addition, this Solve method overload handles secondary columns. The difference between primary and secondary columns is that a solution covers every primary column exactly once but covers every secondary column at most once.
Examples
Copy
var data = new List<Tuple<int[], string>>
    {
        Tuple.Create(new[] {1, 0, 0, 1, 0}, "Some data associated with row 0"),
        Tuple.Create(new[] {0, 1, 0, 0, 0}, "Some data associated with row 1"),
        Tuple.Create(new[] {0, 0, 1, 0, 0}, "Some data associated with row 2")
    };
var dlx = new Dlx();
const int numPrimaryColumns = 3;
var solutions = dlx.Solve(data, d => d, r => r.Item1, numPrimaryColumns);
